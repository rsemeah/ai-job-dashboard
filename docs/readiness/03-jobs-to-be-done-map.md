# 03. Jobs To Be Done Map

## Core functional job
"When I find a promising role, help me produce a credible, tailored application package fast."

## End-to-end JTBD chain
1. **Discover** role opportunity.
2. **Ingest** job posting and context.
3. **Assess fit** using evidence.
4. **Generate** targeted resume + cover letter.
5. **Improve quality** (clarity, ATS compatibility, evidence strength).
6. **Export and apply.**
7. **Prepare interview follow-up.**

## Emotional jobs
- Reduce anxiety and decision fatigue.
- Increase confidence that submitted materials are credible.

## Social jobs
- Present as prepared, intentional, and role-aware.

## Current support level (initial assessment)
- Discover/Ingest: present.
- Fit/Generate: partially present.
- Improve/Export/Prep: partially present.
- Feedback loops + longitudinal learning: missing/partial.

## Missing
- Decision criteria per stage (what qualifies as "good enough" output).
- Persona-specific flow variants.
