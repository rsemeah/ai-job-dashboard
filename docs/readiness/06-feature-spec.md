# 06. Feature Spec (Current + Planned)

## Core features
1. Job intake and parsing
2. Fit scoring and evidence mapping
3. Resume generation
4. Cover letter generation
5. Interview prep generation
6. Export workflows

## Supporting features
- Profile management
- Templates
- Logs / processing visibility
- Ready queue / applications tracking

## Trust features
- Evidence-match view
- Red-team review page
- Account-scoped data model via Supabase auth

## Quality requirements
- Clear loading/error/empty states
- No placeholder leakage in production UX
- Deterministic handling for partial parse outcomes

## Explicit exclusions (for current scope)
- Team collaboration
- Recruiter marketplace
- Mobile-native app

## Missing spec details
- Per-feature acceptance criteria and SLAs
- Definition of done by feature
- Ownership (who owns product/engineering QA sign-off)
