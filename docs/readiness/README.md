# HireWire Readiness Docs Pack

This folder captures the 12 planning/operating documents needed to move from "code exists" to "launchable, measurable product."

## How to use this pack

1. Treat each file as a living source of truth.
2. Keep the **Captured Now** sections grounded in current repo reality.
3. Move unresolved items into each file's **Open Questions / Missing** section.
4. Update these files alongside product and code changes.
